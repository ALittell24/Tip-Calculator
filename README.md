# Tip Calculator

## Overview

The Tip Calculator is a Python application that helps users calculate restaurant tips and split the total bill among multiple people.

## Features

- Calculate tip percentage
- Split a bill between multiple people
- Display the amount each person should pay
- Simple command-line interface

## Technologies Used

- Python

## Concepts Practiced

- Variables
- User input handling
- Data type conversion
- Arithmetic operations
- String formatting

## What I Learned

This project helped me gain experience working with user input, performing calculations, and presenting results clearly to users. It also strengthened my understanding of Python variables and mathematical operations.
